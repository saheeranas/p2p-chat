## Tech

Technical Information about the project:

### Features

    - P2P 
    - Private 
    - Render ready

### Stack

    - React (Vite + TypeScript)
    - Express
    - PeerJs

### Run the project

1. Build the project

   ```
   npm build
   ```

2. Start the server
   ```
   npm start
   ```
