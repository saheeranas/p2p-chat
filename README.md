## Tech

Technical Information about the project:

### Features

    - P2P
    - PWA
    - Private
    - Heroku ready

### Stack

    - React (CRA + TypeScript)
    - Express
    - PeerJs

### Run the project

1. Build the project

   ```
   npm build
   ```

2. Start the server
   ```
   npm start
   ```
3. Go to http://localhost:5000 in browser to use the app
